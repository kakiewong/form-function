# form-function
A modern furniture company
<br>
<br>
Built with React and React hooks. This project inspired by the Crate & Barrel homepage has many built in features you would find in e-commerce websites, such as appointment booking and a cart feature. Uses React Router for routing, useContext for state management, and Formik for form validation.
